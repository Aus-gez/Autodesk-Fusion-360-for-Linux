Welcome to the Autodesk Fusion 360 for Linux Setup Wizard
Many thanks to you for deciding to use my setup wizard to be able to use Autodesk Fusion 360 on your computer.
This quick setup wizard will help you configure the basic settings and install the program. Furthermore, it is possible to install some tested plugins after the installation.
Depending on your current environment, setup may involve:
- Checking your system for minimum installation requirements.
- It is recommended that you close all other applications before continuing.
Click Next to continue, or Cancel to exit the Setup Wizard.
